# Ghost Investor AI

AI-powered lead enrichment and investor outreach orchestration platform. Designed to integrate with orchestrator systems via REST API.

## Features

### Core Modules

- **Lead Import**: CSV, manual, and API-based lead imports
- **Enrichment Adapters**: Apollo, Clearbit, People Data Labs integration
- **Contact Scoring**: Multi-factor lead quality scoring
- **Email Service Integration**: Gmail and Outlook account management with OAuth2
- **AI Email Generation**: GPT-4 powered personalized email generation (first-touch, follow-up, re-engagement)
- **Reply Parsing**: LLM-based classification and sentiment analysis of incoming replies
- **Outreach Sequences**: Campaign management with follow-up cadence
- **Activity Logging**: Full interaction tracking with CRM sync
- **Batch Operations**: Parallel processing via Celery + Redis (enrich, generate, send, classify)
- **GhostCRM Sync**: Bi-directional synchronization of leads and activities
- **Investor Profiles**: Investment stage, sectors, and preference data

### Phase 1 Capabilities (✅ COMPLETE)

- **Email sending** through Gmail or Outlook with OAuth2 authentication
- **AI-generated personalized emails** using OpenAI GPT-4 with cost tracking
- **Automatic reply classification** (INTERESTED, NOT_INTERESTED, LATER, QUESTION, UNSUBSCRIBE, UNCLEAR)
- **Rate limiting** (50 emails/hour, 500 unique recipients/day) with sender reputation protection
- **Batch job queue** for processing 100s of leads/emails in parallel
- **Bi-directional CRM sync** pushing activities and pulling lead updates
- **Email tracking** (opens, clicks, replies)
- **JWT authentication** with refresh tokens

### MVP Outputs

- Enriched lead records with company and contact data
- Personalized invitational emails generated by AI
- Reply classification with sentiment analysis
- Configurable follow-up cadence (default: immediate, 2 days, 5 days)
- Investor profile summary
- Activity timeline and engagement metrics
- Email delivery and engagement tracking

## API Overview

### Leads
- `POST /api/leads/` - Create new lead
- `GET /api/leads/` - List leads
- `GET /api/leads/{lead_id}` - Get lead details
- `PUT /api/leads/{lead_id}` - Update lead
- `DELETE /api/leads/{lead_id}` - Delete lead
- `POST /api/leads/import/csv` - Bulk import from CSV

### Enrichment
- `POST /api/enrichment/enrich/{lead_id}` - Trigger enrichment
- `GET /api/enrichment/score/{lead_id}` - Get contact score
- `POST /api/enrichment/score/{lead_id}/recalculate` - Recalculate score

### Campaigns
- `POST /api/campaigns/` - Create campaign
- `GET /api/campaigns/{campaign_id}` - Get campaign details
- `POST /api/campaigns/{campaign_id}/add-lead/{lead_id}` - Add lead with generated email
- `POST /api/campaigns/{campaign_id}/schedule` - Schedule campaign sends
- `POST /api/campaigns/{campaign_id}/pause` - Pause campaign
- `POST /api/campaigns/{campaign_id}/resume` - Resume campaign

### Activities
- `POST /api/activities/` - Log activity
- `GET /api/activities/lead/{lead_id}` - Get lead timeline
- `POST /api/activities/{activity_id}/sync-crm` - Sync to CRM

### Email Accounts (Phase 1 ✅)
- `POST /api/email-accounts/authorize/gmail` - Get Gmail OAuth authorization URL
- `POST /api/email-accounts/authorize/gmail/callback` - Complete OAuth2 flow
- `GET /api/email-accounts/` - List authorized email accounts
- `DELETE /api/email-accounts/{account_id}` - Remove email account
- `POST /api/email-accounts/{account_id}/test-connection` - Test email connection

### Batch Operations (Phase 1 ✅)
- `POST /api/batch/enrich-leads` - Submit batch enrichment job (100s of leads)
- `POST /api/batch/generate-emails` - Submit batch email generation job
- `POST /api/batch/send-emails` - Submit batch email sending job
- `POST /api/batch/launch-campaign` - Launch campaign (batch send all prepared emails)
- `GET /api/batch/job-status/{job_id}` - Check background job status

## Quick Start

### Prerequisites
- Python 3.10+
- PostgreSQL 13+
- Redis 7+ (for batch job queue)
- API keys:
  - **Required**: OpenAI (for email generation), Gmail/Outlook OAuth (for email sending)
  - **Optional**: Apollo, Clearbit, People Data Labs (for enrichment)

### Installation

1. **Clone repository**
   ```bash
   git clone https://github.com/burchdad/ghost-investor-ai.git
   cd ghost-investor-ai
   ```

2. **Create virtual environment**
   ```bash
   python -m venv venv
   source venv/bin/activate  # On Windows: venv\Scripts\activate
   ```

3. **Install dependencies**
   ```bash
   pip install -r requirements.txt
   ```

4. **Configure environment**
   ```bash
   cp .env.example .env
   # Edit .env with your:
   # - OPENAI_API_KEY
   # - GMAIL_CLIENT_ID & GMAIL_CLIENT_SECRET (or OUTLOOK_CLIENT_ID & OUTLOOK_CLIENT_SECRET)
   # - JWT_SECRET_KEY
   # - Celery broker/backend URLs (Redis)
   ```

5. **Initialize database**
   ```bash
   alembic upgrade head
   # Or: python -c "from src.ghost_investor_ai.database import Base, engine; Base.metadata.create_all(bind=engine)"
   ```

6. **Run services** (in separate terminals):
   ```bash
   # Terminal 1: FastAPI server
   uvicorn src.ghost_investor_ai.main:app --reload
   
   # Terminal 2: Celery worker (for background jobs)
   celery -A src.ghost_investor_ai.services.batch_jobs worker --loglevel=info
   ```

   API will be available at `http://localhost:8000`
   Docs at `http://localhost:8000/docs`

### Docker Setup (Recommended)

```bash
docker-compose up -d

# Services will start:
# - PostgreSQL at localhost:5432
# - Redis at localhost:6379
# - FastAPI at http://localhost:8000
# - Celery worker for batch jobs
```

### Verify Installation

```bash
# Check API health
curl http://localhost:8000/health

# View API docs
open http://localhost:8000/docs
```

## Architecture

### Database Schema

- **leads** - Lead records with enrichment data
- **contact_scores** - Detailed scoring breakdown
- **outreach_emails** - Generated emails
- **outreach_campaigns** - Campaign metadata
- **follow_up_emails** - Follow-up sequence emails
- **activities** - Interaction log
- **investor_profiles** - Investment preference data

### Service Layer

- `EnrichmentService` - Coordinates enrichment adapters
- `ContactScoringService` - Multi-factor scoring logic
- `EmailDraftingService` - Email generation
- `OutreachSequenceService` - Campaign and sequence management
- `ActivityLoggingService` - Activity tracking and CRM sync
- `LeadImportService` - Lead ingestion

### API Design

- RESTful API with FastAPI
- Pydantic models for validation
- Async/await for non-blocking operations
- Background tasks for long-running operations
- Health check endpoint for orchestrator integration

## Orchestrator Integration

This service is designed to be called by an AI mission control orchestrator:

```bash
# Health check
curl http://localhost:8000/health

# Create lead
curl -X POST http://localhost:8000/api/leads/ \
  -H "Content-Type: application/json" \
  -d '{"email": "investor@example.com", "first_name": "John", ...}'

# Trigger enrichment
curl -X POST http://localhost:8000/api/enrichment/enrich/1

# Get contact score
curl http://localhost:8000/api/enrichment/score/1

# Create campaign
curl -X POST http://localhost:8000/api/campaigns/ \
  -H "Content-Type: application/json" \
  -d '{"name": "Q1 Outreach", "description": "..."}'

# Add lead to campaign with email
curl -X POST http://localhost:8000/api/campaigns/1/add-lead/1

# Schedule campaign
curl -X POST http://localhost:8000/api/campaigns/1/schedule
```
